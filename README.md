# Family Task Planner 📋

Een takenlijstapp voor families met QR-code scanning en AI-planning.

## Features

- ✅ Takenlijst beheren (toevoegen, bewerken, verwijderen)
- ✅ QR-code scanning voor taken markeren als afgerond
- ✅ AI-gestuurde taakplanning (OpenAI)
- ✅ Familie-samenwerking
- ✅ Responsive design

## Project Structuur

```
family-task-planner/
├── frontend/          (React + TypeScript)
├── backend/           (Node.js + Express)
├── README.md
└── .gitignore
```

## Quick Start

### Backend starten
```bash
cd backend
npm install
npm run dev
```

### Frontend starten
```bash
cd frontend
npm install
npm start
```

## Tech Stack

- **Frontend**: React 18, TypeScript, Axios
- **Backend**: Node.js, Express, SQLite3
- **QR Codes**: qrcode library
- **AI**: OpenAI API
